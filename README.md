---

## Current Project — Network Monitor

A Python-based network monitoring tool that:

- Detects new devices via MAC address and IP
- Tracks online/offline status with continuous scans
- Logs connection and disconnection events with timestamps
- Detects ARP spoofing attempts *(in progress)*
- Web-based dashboard accessible via localhost

**Status:** Active development — first release coming soon.
